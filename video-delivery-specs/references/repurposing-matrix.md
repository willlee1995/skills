# Repurposing Matrix & Reflow Recipe

Goal: one master, every ratio, no broken layouts. Decide the strategy before animating.

## Full matrix

| From → To | Method | Effort | Watch for |
|---|---|---|---|
| 16:9 → 4:5 | Light side-crop to 1080 wide | Low | Centre the action; minor edge loss |
| 16:9 → 1:1 | Crop sides to centre 1080×1080 | Low-med | Text/logos near L/R edges cut |
| 16:9 → 9:16 | Crop to centre + re-stack vertically | High | Wide comps collapse; reflow needed |
| 16:9 → 2.39:1 | Letterbox or re-frame | Low | Lose top/bottom; reframe key action |
| Master → 15s cut | Keep hook + payoff, drop middle | Med | Re-time transitions, keep brand+CTA |
| Master → 6s bumper | One beat: hook OR payoff + logo | Med | Must still read at a glance, muted |

## The centre-spine method (plan-ahead)

When animating the 16:9 master, keep all essential content — message, key product, logo, CTA — inside a centred **1080×1080 spine**. Every crop (1:1, 4:5, 9:16-core) then keeps the message automatically. Background and supporting motion can live in the wider area.

## 16:9 → 9:16 reflow recipe (the hard one)

A hard crop rarely works for wide compositions. Reflow instead:

1. **Re-canvas** to 1080×1920; place the master's centre spine in the safe core.
2. **Re-stack horizontally-arranged elements vertically** (a left-right split becomes top-bottom).
3. **Scale up the focal element** — vertical has less width; small wide-frame text becomes unreadable.
4. **Move titles into the caption-safe band** (clear of top 250 / bottom 400 / TikTok right column).
5. **Re-time if needed** — vertical attention is faster; tighten holds.
6. **Re-pin the logo/CTA** to a safe position, not the cropped edge.
7. **Check muted legibility** — most vertical viewing is sound-off.

Budget this as real work (often 0.5-1.5 days per master for a clean 9:16), not a one-click export.

## Cutdown timing rule

Do not just trim the master to length. Re-edit:
- Lead with the strongest hook in the first 1-2 seconds.
- Preserve the single core message and the brand/CTA.
- Recut transitions so the shorter version flows, not stutters.
- A 6s bumper should land one idea, readable on mute, with the logo.

## Export set checklist (per project)

- [ ] 16:9 master (high bitrate / ProRes)
- [ ] 9:16 (reflowed, not cropped) for Reels/TikTok/Shorts
- [ ] 1:1 and/or 4:5 for Feed
- [ ] Cutdowns at each required length
- [ ] Captioned (burned-in) social versions
- [ ] Clean/textless master if broadcast or future edits expected
- [ ] Static end-frame / thumbnail if specified
